# Jogo do Mario - Mario Jump Game

Um jogo de plataforma infinito onde você controla o Mario pulando sobre obstáculos e coletando moedas em 5 fases diferentes.

## Como Jogar

- **Pular:** Espaço, W ou Seta para Cima
- **Mover:** A/D ou Setas Esquerda/Direita
- **Pausar:** ESC ou P

## Características

### 5 Fases Únicas
1. Mundo Verde - Grassland clássico
2. Caverna - Mundo subterrâneo
3. Castelo - Mundo de lava
4. Oceano - Mundo aquático
5. Céu - Mundo das nuvens

### Power-ups
- Super Cogumelo (+1 vida)
- Super Estrela (invencibilidade 8s)
- Sonic (super velocidade 10s)

## Tecnologias

- HTML5, CSS3, JavaScript ES6+
- Web Audio API
- requestAnimationFrame (60 FPS)
- localStorage

## Como Executar

1. Clone o repositório
2. Abra `index.html` no navegador
3. Pressione JOGAR

## Performance

- 60 FPS estáveis
- Zero vazamentos de memória
- Código modular e organizado
